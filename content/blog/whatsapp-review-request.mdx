## 이메일 vs SMS vs WhatsApp 리뷰 요청

리뷰 요청 채널별 평균 전환율:

- **이메일**: 5-10% (스팸 필터, 낮은 오픈율)
- **SMS**: 15-20% (비용 부담, 법적 제한)
- **WhatsApp**: 20-25% (높은 오픈율, 친근한 채널)

WhatsApp은 한국에서도 점점 비즈니스 커뮤니케이션 도구로 자리잡고 있습니다.

## WhatsApp 리뷰 요청 모범 사례

### 1. 메시지 타이밍

서비스 완료 후 **2-4시간 이내**가 최적입니다. 너무 빠르면 부담, 너무 늦으면 기억이 희미해집니다.

### 2. 메시지 작성 팁

- 고객 이름을 포함한 개인화된 메시지
- 간단한 감사 인사 + 리뷰 링크
- 1분이면 끝난다는 점 강조

### 3. 리뷰 링크 최적화

직접 리뷰 작성 화면으로 연결되는 단축 URL을 사용하세요. ReviewHub의 QR 코드 기능을 활용하면 자동으로 최적화된 링크가 생성됩니다.

## ReviewHub WhatsApp 통합 활용법

ReviewHub에서 WhatsApp 리뷰 요청을 보내는 방법:

1. Review Requests 페이지에서 "Send Request" 클릭
2. 고객 이름과 전화번호 입력
3. 플랫폼 선택 (Google, Yelp 등)
4. 자동으로 개인화된 WhatsApp 메시지 발송

모든 요청의 발송/읽음/완료 상태를 실시간으로 추적할 수 있습니다.
